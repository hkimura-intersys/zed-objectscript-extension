# zed-objectscript-extension
An [ObjectScript Routines](https://docs.intersystems.com/irislatest/csp/docbook/DocBook.UI.Page.cls?KEY=GCOS_intro) extension for [Zed](https://zed.dev/) to support development for the InterSystems IRIS product.

## Features
### Syntax Highlighting
This Zed extension uses the [tree-sitter-objectscript](https://github.com/intersystems/tree-sitter-objectscript) grammar to provide syntax highlighting and code injections for `.rtn`, `.mac`, `.int`, and `.inc` files containing ObjectScript.
