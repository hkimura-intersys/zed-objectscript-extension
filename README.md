# zed-objectscript-extension
An [ObjectScript](https://docs.intersystems.com/irislatest/csp/docbook/DocBook.UI.Page.cls?KEY=GCOS_intro) extension for [Zed](https://zed.dev/) to support development for the InterSystems IRIS product.

## Features
### Syntax Highlighting
This Zed extension uses the [tree-sitter-objectscript](https://github.com/intersystems/tree-sitter-objectscript) grammar to provide syntax highlighting and code injections for .cls files containing ObjectScript.

### Diagnostics
- For a superclass, click on a method name to get a list of an implementation of this method in subclasses.
